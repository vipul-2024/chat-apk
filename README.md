# server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

1-bun install
2-bun init
3-express install (npm install express)
4-dockerfile(npm i doker)
5-bun run dev
